## Building from source

```
$ git clone https://github.com/bootifulmicropizza/_project bootifulmicropizza
$ cd bootifulmicropizza
$ git submodule init
$ git submodule update
$ tools/build.sh
```

## Running locally

```
$ tools/run_local.sh
```
